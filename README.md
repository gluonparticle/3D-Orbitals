## Description

The simulation:

- Computes radial and angular components of 4f orbitals
- Generates a probability density distribution
- Samples electron positions based on the normalized probability
- Produces a 3D scatter visualization of the orbital shape

---

## Tech Stack

- Python  
- NumPy  
- Pandas  
- Matplotlib (3D plotting)

---

The resulting figures demonstrate spatial electron distributions for different 4f orbital configurations of Cerium.
